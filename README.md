# PornHub 视频下载器使用文档

## 软件介绍

PornHub 视频下载器是一个功能强大的GUI工具，用于下载 PornHub 网站上的视频。它支持以下功能：

- 单视频下载：解析单个视频链接，获取不同分辨率并下载
- 批量下载：同时处理多个视频链接，自动批量下载
- 多种分辨率：支持获取和选择不同画质的视频
- 现代化界面：美观易用的图形界面
- 实时进度：显示下载进度和状态
- 错误处理：完善的错误提示和日志记录

## 系统要求

- Windows 7/8/10/11
- Python 3.7+（仅源码运行需要）
- 稳定的网络连接

## 安装方法

### 方法一：使用可执行文件（推荐）

1. 打开 `dist` 文件夹
2. 找到并双击 `PornHubDownloader.exe` 运行
3. 无需安装其他依赖，直接使用

### 方法二：从源码运行

1. 确保已安装 Python 3.7+
2. 安装依赖：
   ```
   pip install -r requirements.txt
   ```
3. 运行程序：
   ```
   python main.py
   ```

## 使用指南

### 单视频下载

1. **选择模式**：在"下载模式"下拉菜单中选择"单视频下载"
2. **输入链接**：在输入框中粘贴 PornHub 视频链接
3. **解析视频**：点击"解析"按钮，程序会自动获取视频信息和可用分辨率
4. **选择分辨率**：从下拉菜单中选择想要的视频分辨率
5. **选择保存路径**：点击"浏览"按钮选择视频保存位置
6. **开始下载**：点击"下载"按钮开始下载视频
7. **查看结果**：下载完成后会弹出提示，并在日志中显示保存位置

### 批量下载

1. **选择模式**：在"下载模式"下拉菜单中选择"批量下载"
2. **输入链接**：在文本框中输入多个视频链接，每行一个
3. **选择保存路径**：点击"浏览"按钮选择视频保存位置
4. **开始批量下载**：点击"开始批量下载"按钮
5. **查看进度**：程序会自动处理每个视频，显示实时进度
6. **查看结果**：批量下载完成后会弹出提示，显示下载结果

## 功能特点

- **自动解析**：智能解析视频链接，获取可用分辨率
- **多分辨率支持**：支持 1080p、720p、480p 等多种画质
- **批量处理**：一次处理多个视频链接，节省时间
- **错误重试**：自动处理网络错误，提高下载成功率
- **日志记录**：详细的操作日志，便于问题排查
- **用户友好**：直观的界面设计，操作简单

## 常见问题

### 1. 解析失败

- **原因**：视频链接无效或网络连接问题
- **解决方法**：检查链接是否正确，确保网络连接正常

### 2. 下载失败

- **原因**：网络不稳定或服务器限制
- **解决方法**：尝试重新下载，或更换网络环境

### 3. 视频质量选择

- **说明**：批量下载模式会自动选择最高分辨率
- **单视频模式**：可以手动选择想要的分辨率

### 4. 保存路径问题

- **说明**：确保选择的保存路径有写入权限
- **解决方法**：选择一个有写入权限的目录，如用户下载文件夹
